# Aula 2: Experimentação e MVP de Modelos

## Descrição
Nesta aula você aprenderá a realizar experimentos com modelos de Machine Learning e rastrear resultados usando MLFlow.

## Objetivos
- Configurar e usar MLFlow para rastreamento de experimentos
- Experimentar com diferentes algoritmos e hiperparâmetros
- Comparar resultados de múltiplos experimentos
- Criar um MVP (Minimum Viable Product) de modelo

## Exercício
Execute o notebook `exercicio_experimentacao_mvp.ipynb` para praticar os conceitos.

## Como usar
```bash
jupyter notebook exercicio_experimentacao_mvp.ipynb
```

## Para visualizar experimentos no MLFlow UI
```bash
mlflow ui
```
Acesse http://localhost:5000
